# CTF-tools

## Usage

### local

#### init the python venv

    python3 -m venv .venv && source ./venv/bin/activate && pip install -r requirements.txt


### using docker

#### init the container

    ./docker-init.sh

#### run start the container

    ./docker-init.sh

#### when the container is up run

    ./init-python-in-docker.sh
